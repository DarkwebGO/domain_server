Le niveau de sécurité de votre Navigateur Tor est trop faible. Veuillez utiliser le bouton de la barre d’outils de votre navigateur pour le changer.

Comment changer votre niveau de sécurité

  1. Cliquez sur le bouton dans la barre d’outils ci-dessus
  2. Cliquez sur Changer pour ouvrir les préférences de niveau de sécurité
  3. Sélectionnez Le plus sûr et fermer l’onglet des préférences

Actualisez cette page et vous aurez terminé.
