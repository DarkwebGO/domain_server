hectoByte

Published: 2012-08-10 in Demoscene

Prod pages: [pouet.net] [demozoo.org]
Date: 2012-08-10
Party: Outline 2012
Compo: Combined 128b
Platform: Atari VCS
Rank: 2nd

Tried to get as much variation as possible into a 128 byte entry. A respectable 2nd place, considering fsqrt was in the compo (and won).

                                  hectoByte

      ~~ A VCS intro needs neither VSYNC nor proper interrupt vectors ~~

     Works best when run on real hardware and hooked up to a forgiving TV,
loaded onto a Harmony cart's EEPROM ("harmonycart hectoByte.bin" using the CLI)

      In Stella, run hectoByte-stella.bin or it runs at incorrect speed

  The Harmony menu and MESS don't support 128 B - use hectoByte-4k.bin instead
